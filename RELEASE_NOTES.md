## v3.0.0 – Clean UI Overhaul (2025-11-17)

### Highlights
- Unified dashboard layout with clearer separation:
  - Sources (left)
  - Configuration (right)
  - Draft Output
  - Versions timeline
  - Future roadmap
- Dynamic top-level page titles based on tab (Dashboard, Projects, etc.)
- Redesigned first-time “getting started” panel above dashboard
- Revised export block below editor with Copy, TXT, DOC (.PDF coming soon)
- Toast notifications for all important actions
- Score pill colour logic (good / average / poor / default)
- Sidebar cleaned and simplified
- Smoothed typography, card spacing and visual consistency
- New Project action now resets the workspace and returns to Dashboard
- Dead code removed and structure cleaned for easier iteration
